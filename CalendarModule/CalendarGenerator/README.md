# Calendar Generator

A simple Python program that displays a calendar in the terminal using the built-in
calendar module.

The user can view either the full calendar of a year or a specific month.

## What it does

- Takes a year as input
- Shows the full year calendar or one month
- Starts the week from Sunday
- Handles invalid inputs
- Allows the user to restart or exit
