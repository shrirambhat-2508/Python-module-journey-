Dice Roller - Press Enter to Roll

Description:
This is a simple Python dice rolling game using the random module. Each time you press Enter, the program rolls a dice and displays a random number between 1 and 6. If you roll a 6, the program congratulates you for rolling the lucky number.

How to Play:
1. Run the dice_roller.py file.
2. Press Enter to roll the dice.
3. If you roll a 6, enjoy the celebration message!
4. After each roll, the program asks if you want to roll again.
5. Type 'yes' to roll again or 'no' to quit.

Example Run:

Welcome to Dice Roller!

Press Enter to roll the dice...
You rolled: 4

Press Enter to roll the dice...

Congratulations! You rolled the lucky 6!

Tech Used:
