Rock-Paper-Scissors - Best of 3

Description:
This is a simple Python game where you play Rock-Paper-Scissors against the computer. The game is played in best of 3 rounds. Instead of typing the choice names, you select a number:
1 = Rock
2 = Paper
3 = Scissors
The computer randomly selects its choice, and the winner of each round is decided based on standard rules:
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

How to Play:
1. Run the rps.py file.
2. Enter your choice using numbers (1, 2, or 3).
3. After 3 rounds, the scores are displayed and the winner is announced.
4. You will be asked if you want to play again.

Features:
- Input validation ensures only valid numbers are accepted.
- Best of 3 rounds for more engaging gameplay.
- Option to replay the game after each session.
